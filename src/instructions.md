Here are the instructions to run the Python Code:

Setup Azure OpenAI service
1.	Create Azure OpenAI service if not already created
2.	Deploy completions OpenAI model if not already done
3.	Fill up the .env file. A template can be found as .env.template. Paste the file in the src folder.

Install the requirements by running
`Pip install -r requirements.txt`

Now, we are good to run the evaluate function in src/evaluator/evaluator.py
